Mac 下用 Latex 写简历

发表于 2019-04-11 更新于 2021-06-26

Mac 推荐安装的是 Mactex，通过 brew search latex 可以找到三种:

  • mactex: 完整安装，包括编辑器和完整的库，安装完需要 6G+
  • mactex-no-gui: 同上，只是没有 gui 的编辑器
  • basictex: 基础包，100M 以内；没有对应的依赖库

如果为了省心，推荐第一种；为了省空间推荐最后一种。
