rewrite 模块启用的 Forbidden 问题

发表于 2008-08-20 更新于 2021-06-29 分类于 ubuntu

问题：如果要支持 rewrite，必须在设置目录权限的时候使用 AllowOverride All，但是我一访问就提示 403 错误（Forbidden），提示你没有权限访问该目录

原来要使用 rewrite 模块，不仅仅要 AllowOverride All，还必须要设置 Options FollowSymLinks。

这样才能成功使用。

在 ubuntu 启动 rewrite 的命令：sudo a2enmod rewrite

ubuntu 下命令行发送飞信（成功）

发表于 2008-08-13 更新于 2021-06-29 分类于 ubuntu

在 ubuntu 下可以用命令行发飞行短信了，以后可以配置个，发送些日志消息或报警短信到自己手机上了。

测试环境：ubuntu 8。10  （X86/32）

官网地址：http://www.it-adv.net/

在网站上下载支持库和最新程序（LINUX X86/32），其他版本请到官方网站下载。

首先配置支持库，解压你下载的支持库。包含四个文件：

libACE.so.5.4.7  libACE_SSL.so.5.4.7  libcrypto.so.0.9.8  libssl.so.0.9.8

使用命令：cp *  /usr/lib/
复制到用户的库目录下。

然后解压你下载的程序文件，fetion 就是主程序。

./fetion -u 帐号 -p 密码

这样就成功运行了。

1 插件开发原理

系统运行时，当遇到特写的事件，就会自动调用 plugins 目录的相应程序。我们可以根据实际应用来重写这些程序来达到自己的要求。

目录，系统支持四类插件：

1、定时插件：系统每 10 秒调用一次 plugins/plugin_timer
自带演示插件的功能：每隔 10 秒钟更换一次心情。

2、消息插件：系统收到消息时，调用 plugins/plugin_message
自带演示插件的功能：发送反话给对方。比如说，对方和你讲 ABCD。插件自动发回 DCBA。

3、激活对话窗口插件：当对方双击，打开对话窗口时，激活该插件 plugin_invite。
该功能适合做机器人。当对方激活后，立即提示帮助信息。帮助插件中带的是演示。

4、好友申请插件：当对方增加您为好友时，激活该插件 plugin_buddy_application。
程序中存在 2 处好友申请开关，插件的优先级高于全局的开关（通过 autoaccept on/off 设置的值）

插件的详细说明可以参见：http://it-adv.apt-get.cn/bbs/viewthread.php?tid=28 windows 下的使用方法参见：http://it-adv.apt-get.cn/bbs/viewthread.php?tid=7&extra=page%3D1

LibFetion 飞信 API

发表于 2008-08-08 更新于 2021-06-29 分类于 其他

LibFetion 是一款支持多操作系统，使用 C/C++ 语言，针对中国移动 IM 飞信通信协议开发的第三方应用程序库。

LibFetion 封装了飞信通信协议并存储相关数据，给开发者提供简洁、丰富的 API 接口。使用 LibFetion 飞信客户端的开发者只需专注于 GUI (图形用户界面) 的设计开发，因此大大提高飞信客户端的开发速度和降低开发难度。使得飞信在多个操作系统平 台上运行，达到 “fetion anywhere“。

LibFetion 是针对嵌入式设备系统设计的，它只使用了 C 库，提供平台线程库接口（缺省使用 posix 线程库) 和网络接口（缺省使用 socket）因此具有较强的可移植性。目前 LibFetion 已在 Linux，Mac OS X，Windows，ucLinux，Ecos 操作系统上成功运行。

如今基于 LibFetion 开发的 PC 客户端程序 LibFx 已经发布 V0.8.0 版了，点击下载。

官方网址：http://www.libfetion.cn/index.html

ubunt 下 wine＋TM2008 Preview4 稳定运行

发表于 2008-07-29 更新于 2021-06-29 分类于 ubuntu

在 czk 的指导下成功运行了 TM2008 Preview4，而且很稳定。

成功运行：

1. 消息的正常传输，包括聊天中的图篇都能正常显示

2. 可以进行截图，虽然不能有半透明的选择框，但是满足不精确截屏是没有问题了（考验你的眼力了）

3. 发送文件正常，发送一般的文件都正常。大小好像也没有影响，支持短点，哈哈

4. 更改签名，更改备注等都没有问题

目前就以上能实现吧，但是比 eva 要强大多了。

安装：

1. 安装 wine，通过库可以直接安装，现在版本是 wine-1.1.2，
sudo apt-get install wine
你可以用 wine --version 来查看版本。

2. 下载 TM2008 Preview4，一定要是这个版本，试了下 beat 好像不行，用这个就满足需求了。 下载：TM2008 Preview4. 你可以英文版

3. 安装 tm：

4. 跳到你下载的路径下，执行 wine tm2008preview4_sp2_chs.exe

5. 安装完成后，并不能运行，缺少一些 dll。
将 tm2008 目录下 SysDir 目录下的各个子目录（Microsoft.VC80.ATL 等 5 个子目录）下的所有 dll 文件拷贝到 tm2008 下的 Bin 目录中。（如果你没有 SysDir 目录，这里下载）

6. 这样你就 可以运行了。一般桌面会产生一个快捷图标的。

如果遇到中文乱码，注意做如下配置：

1. 如果用的英文 locale，记得改成中文再启动 wine。即 export LANG=zh_CN.UTF-8

2. 配置中文字体。将 windows 下的 simsun.ttc 拷贝到～/.fonts/ 目录下，然后运行 regedit，修改
[HKEY_CURRENT_USER\Software\\Wine\\Fonts\\Replacements]
"Tahoma"="宋体"

移除 /usr/share/wine/fonts/ 下的 tahoma.ttf 和 tahomabd.ttf 的字体文件
