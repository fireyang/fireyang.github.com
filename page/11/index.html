      AmFast 另一个python的AFM3

      发表于 2009-03-24 | 分类于 python

      AmFast


      特色：

      • AmFast的底层的编码和解码采用了 C语言, 因此效率比PyAmf快18倍左右（本人没有测试过）.

      •编码器和解码器需要用户定义Python对象，并允许自定义编码/解码过程。

      • 支持自定义类映射

      • 支持 NetConnection 和 RemoteObject进行Remoting.

      • Remoting 头上可以设置标记，快速的实现用户验证等，还有其他AMF的特性

      • 支持 Actionscript代码类的映射
