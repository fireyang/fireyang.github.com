      AmFast 另一个 python 的 AFM3

      发表于 2009-03-24 更新于 2021-06-29 分类于 python

      AmFast

      特色：

      ・AmFast 的底层的编码和解码采用了 C 语言，因此效率比 PyAmf 快 18 倍左右（本人没有测试过）.
      ・编码器和解码器需要用户定义 Python 对象，并允许自定义编码 / 解码过程。
      ・支持自定义类映射
      ・支持 NetConnection 和 RemoteObject 进行 Remoting.
      ・Remoting 头上可以设置标记，快速的实现用户验证等，还有其他 AMF 的特性
      ・支持 Actionscript 代码类的映射
