发表于 2009-03-28 更新于 2021-06-26 分类于 杂七杂八

（出现在项目的侧边栏上）

原来一发现有好的项目都用书签的方式，收藏到 google 书签，现在好了，可以直接标注了

这样一进入 code.google.com 的 Project Hosting，在 My projects 就可以看到共享的项目了

真是太方便了
